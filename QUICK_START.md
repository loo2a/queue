# دليل البدء السريع

## في أقل من 10 دقائق!

### الخطوة 1: إعداد Firebase (5 دقائق)
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/)
2. أنشئ مشروع جديد
3. اختر "Realtime Database"
4. أنشئ قاعدة بيانات في وضع الاختبار
5. اذهب إلى "Project Settings" وانسخ بيانات التكوين

### الخطوة 2: تعديل الملفات (3 دقائق)
1. في كل ملف HTML (admin.html, display.html, control.html, tickets.html)
2. ابحث عن `YOUR_API_KEY_HERE`
3. استبدلها ببياناتك من Firebase

### الخطوة 3: الرفع إلى GitHub (2 دقيقة)
1. أنشئ repository جديد على GitHub
2. رفع جميع الملفات
3. فعّل GitHub Pages من الإعدادات

### والآن جاهز للاستخدام!

## الاختبار المحلي
```bash
# لتشغيل الخادم المحلي
python -m http.server 8000

# ثم افتح في المتصفح
http://localhost:8000
```

## أول استخدام
1. افتح `admin.html`
2. أضف عيادتك الأولى
3. اختبر النظام باستخدام `display.html` و`control.html`

## نصائح سريعة
- كلمة المرور الافتراضية للعيادات: `1234`
- جرب نداء رقم معين أولاً
- تأكد من إذن الصوت في المتصفح
- استخدم Chrome للحصول على أفضل أداء
